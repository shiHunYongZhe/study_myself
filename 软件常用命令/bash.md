pwd
ls

echo

alias 'cdmydir=cd /etc/sysconfig'  设置快捷路径
移动光标快捷键
Ctrl + A 光标回到行首（Home）
Ctrl + E 光标回到行尾（End）
Ctrl + F 光标右移一个字符（->）
Ctrl + B 光标左移一个字符（<-）

Ctrl + L：清除屏幕并将当前行移到页面顶部。
Ctrl + C：中止当前正在执行的命令。
Ctrl + Z：暂停当前正在执行的命令。

Shift + PageUp：向上滚动。
Shift + PageDown：向下滚动。
Ctrl + U：删除(剪切)光标位置到行首的字符。
Ctrl + K：删除(剪切)光标位置到行尾的字符。
Ctrl + W：删除(剪切)光标的前一个单词。
Ctrl + H：删除光标的前一个字符。
Ctrl + Y：粘贴已剪切的文本。
Ctrl + D：关闭 Shell 会话。

Ctrl + R：搜索命令行使用过的历史命令记录。
Ctrl + G：退出搜索历史命令记录。
↑，↓：浏览已执行命令的历史记录。
Tab: 自动补全


vi 文件名   用编辑器打开文件
敲击insert  开始编辑修改文件
1、敲击esc, 文件下方的insert消失，输入:wq!   保存修改并退出
                                输入q!   不保存修改并退出
                                输入dG,  删除光标当前行并将光标移至上一行行首，若此时光标在文本开头，则删除全部内容

波浪线~会自动扩展成当前用户的主目录。  echo ~
?字符代表文件路径里面的任意单个字符，不包括空字符  echo ?.txt
如果匹配多个字符，就需要多个?连用。 echo ??.txt  两个字符  echo ???.txt 三个字符
*字符代表文件路径里面的任意数量的任意字符，包括零个字符  echo *.ext
文本文件在子目录，*.txt不会产生匹配，必须写成*/*.txt
方括号扩展的形式是[...]，只有文件确实存在的前提下才会扩展  echo [aeiou].txt  [aeiou]可以匹配五个字母中的任意一个
[!a]表示文件名第二个字符不是a的文件名
方括号扩展有一个简写形式[start-end]，表示匹配一个连续的范围。比如，[a-c]等同于[abc]，[0-9]匹配[0123456789]
大括号扩展{...}表示分别扩展成大括号里面的所有值，各个值之间使用逗号分隔,逗号前后不能有空格   echo d{a,e,i,u,o}g   =>  dag deg dig dug dog
逗号前面可以没有值，表示扩展的第一项为空  cp a.log{,.bak} =>  cp a.log a.log.bak
大括号扩展有一个简写形式{start..end}，表示扩展成一个连续序列(支持逆序)   echo d{a..d}g =>   dag dbg dcg ddg

大括号扩展的常见用途为新建一系列目录。
mkdir {2007..2009}-{01..12}
另一个常见用途，是直接用于for循环
for i in {1..4}
do
  echo $i
done
整数前面有前导0，扩展输出的每一项都有前导0
echo {001..4} 001,002,003,004
